Redux Add
=========

This is just a script I use a lot for generating my initial redux files

Install
-------
```
npm i -g redux-add
```

Usage
-----
```
redux-add [dirname] [name]
redux-add . user
```

Generates:

```
current dir
└───actions
│   │   user.js
└───types
    │   user.js
└───reducers
    │   user.js
```
